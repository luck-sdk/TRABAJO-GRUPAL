Resumen del Proyecto

En el desarrollo de aplicaciones modernas, la gestión eficiente de datos es un componente fundamental. Las herramientas y técnicas de Mapeo Objeto-Relacional permiten conectar el mundo de los objetos en la programación con las bases de datos relacionales.

Durante este proyecto, utilizamos Visual Studio 2022 para el desarrollo del aplicativo web y Microsoft SQL Server como sistema gestor de base de datos (SGBD). Además, reforzamos los fundamentos teóricos de álgebra relacional.

El objetivo principal fue construir un sistema funcional que aplicará los conceptos de diseño de software, lógica relacional y comunicación entre capas de datos mediante OMR.

Puntos clave: Visual Studio 2022, Microsoft SQL server.

Abstract

In modern application development, efficient data management is a key component. Object-Relational Mapping (ORM) tools and techniques bridge the gap between the object-oriented world of programming and relational databases.

Throughout this project, we used Visual Studio 2022 for web application development and Microsoft SQL Server as the database management system (DBMS). We also reinforced the theoretical foundations of relational algebra, which provides the mathematical basis for SQL queries.

The main goal was to build a fully functional system that applies the principles of software design, relational logic, and data layer communication through ORM.

Key points: Visual Studio 2022, Microsoft SQL Server.

1. Introducción

En el entorno de desarrollo Visual Studio 2022, las consultas de selección y proyección desempeñan un papel fundamental en el manejo y visualización de datos dentro de las aplicaciones. Estas consultas permiten extraer información de fuentes de datos como bases de datos, listas o colecciones, facilitando la creación de sistemas más eficientes y organizados.

El uso de estas consultas dentro de Visual Studio 2022 contribuye a optimizar el rendimiento de las aplicaciones, reducir la carga de datos innecesarios y mejorar la presentación de la información en interfaces gráficas o reportes. Por ello, comprender su funcionamiento y aplicación resulta esencial para el desarrollo de software orientado a la gestión de datos.

Consultas de selección

Código de la primera consulta de selección en el "btnConsulta1"

C# Consulta de Selección 1
protected void btnConsulta1_Click(object sender, EventArgs e)
{
    var clientes = from c in db.Customers
                    select new
                    {
                        Codigo = c.CustomerID,
                        Nombre = c.CompanyName,
                        Ciudad = c.City,
                        Pais = c.Country
                    };

    gvConsultas.DataSource = clientes.ToList();
    gvConsultas.DataBind();
}

Código de la segunda consulta de selección.

C# Consulta de Selección 2
protected void btnConsulta2_Click(object sender, EventArgs e)
{
    var bajoStock = from p in db.Products
                            where p.UnitsInStock < 10
                            select new
                            {
                                Producto = p.ProductName,
                                Stock = p.UnitsInStock
                            };

    gvConsultas.DataSource = bajoStock.ToList();
    gvConsultas.DataBind();
}

Código de la tercera consulta de selección.

C# Consulta de Selección 3
protected void btnConsulta3_Click(object sender, EventArgs e)
{
    var empleados = from emp in db.Employees
                            where emp.City == "London"
                            select new
                            {
                                Nombre = emp.FirstName + " " + emp.LastName,
                                Cargo = emp.Title,
                                Ciudad = emp.City
                            };

    gvConsultas.DataSource = empleados.ToList();
    gvConsultas.DataBind();
}

Consultas de proyección

Código de la primera consulta de proyección.

C# Consulta de Proyección 1
protected void btnProyeccion1_Click(object sender, EventArgs e)
{
    var totalPorProducto = from p in db.Products
                                    select new
                                    {
                                        Producto = p.ProductName,
                                        Total = p.UnitPrice * p.UnitsInStock
                                    };

    gvProyecciones.DataSource = totalPorProducto.ToList();
    gvProyecciones.DataBind();

}

Código de la segunda consulta de proyección.

C# Consulta de Proyección 2
protected void btnProyeccion2_Click(object sender, EventArgs e)
{
    var añosPedidos = from o in db.Orders
                                select new
                                {
                                    o.OrderID,
                                    Año = o.OrderDate.HasValue ? (int?)o.OrderDate.Value.Year : null
                                };

    gvProyecciones.DataSource = añosPedidos.ToList();
    gvProyecciones.DataBind();
}
